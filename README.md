# Desafio Tetris Stack - Controle de Peças com Estruturas de Dados

O jogo simula uma versão simplificada do Tretris, onde o jogador começa com uma fila de 5 peças e tem as
opções de jogar ou inserir uma nove peça, lembrando que a fila de peças não deve ultrapassar a quantidade
de 5 peças.

*   O menu principal exibe as opções do jogo.
*   O jogador deve escolher uma opção digitando o respectivo número e pressionando a tecla Enter.
*   O programa irá confirmar a ação e exibir a fila atual.
*   Se o número de jogadas exeder o número de peças a fila ficará vazia.
*   O jogador tem a opção de adicionar paças para continuar jogando.
*   Por fim, a opção de sair do jogo.
